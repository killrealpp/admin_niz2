# Что нужно уточнить в базе знаний

Этот файл — заметка для владельца/директора и администратора. Пока эти пункты не заполнены, бот не должен придумывать точные цены, условия или характеристики.

## 1. Цены

Файл для заполнения: `app/knowledge/prices.md`.

Сейчас достоверно указано:
- Беседка №1 на пляже Максима Горького — 10 500 ₽.
- Предоплата для фиксации брони — 50%.

Нужно уточнить и внести:
- цена Беседки №1 на Русалочке / Беленький песочек;
- цена Беседки №2;
- цена Беседки №3;
- цена Беседки №4;
- цена Беседки №5;
- цена Беседки №6;
- цена Беседки №7;
- цена Беседки №8;
- цена крытой беседки;
- цена тёплой беседки;
- цена летней беседки;
- цена бани;
- цена дома / гостевого дома;
- цена комплекса “беседка + баня”;
- цены допов: уголь, розжиг, решётки, шампуры, посуда, вода, лёд, украшения, уборка, продление.

## 2. Вместимость и удобства

Файл для заполнения: `app/knowledge/objects.md`.

Нужно уточнить:
- вместимость Беседки №1 на пляже Максима Горького;
- точное соответствие номеров беседок локациям;
- где есть свет;
- где есть розетки;
- где есть шторы / жидкие стекла;
- где мангал входит в стоимость;
- какие беседки доступны для короткой аренды, если она вообще возможна;
- точные условия по бане: вместимость, бассейн, минимальная длительность, что входит;
- точные условия по дому: вместимость, заезд/выезд, что входит.

## 3. Оплата и предоплата

Файлы для заполнения:
- `app/knowledge/prices.md`;
- позже `config/payments.yaml` или `.env`, когда выберем механику ЮKassa.

Нужно решить:
- предоплата всегда 50% или фиксированная сумма;
- предоплата считается от полной стоимости или по типу услуги;
- можно ли возвращать предоплату;
- сроки возврата / невозврата;
- когда бронь считается окончательно закреплённой;
- кто отправляет ссылку на оплату до подключения ЮKassa.

## 4. Правила объекта

Файл для заполнения: `information.md` или отдельные файлы в `app/knowledge/`.

Нужно уточнить:
- есть ли туалет на каждой локации;
- есть ли парковка;
- можно ли с животными;
- можно ли купаться;
- можно ли музыка / до какого времени;
- можно ли фейерверки;
- точное время заезда;
- что делать, если клиент приехал раньше;
- что делать, если администратор занят при приезде.

## 5. Фото и материалы

Файл для заполнения: `app/knowledge/objects.md`.

Нужно добавить:
- фото бани;
- фото каждой беседки;
- фото дома;
- фото локаций;
- при необходимости ссылки на альбомы или файлы.

## 6. Важно для бота

Пока цена или условие не заполнены в базе знаний, бот должен отвечать осторожно:

“Точную стоимость/условие уточнит администратор. Я могу собрать заявку и передать её на подтверждение.”

Бот не должен придумывать:
- цены;
- скидки;
- возвраты;
- доступность допов;
- условия оплаты;
- точные характеристики объекта, если их нет в базе.
